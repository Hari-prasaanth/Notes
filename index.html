GitHub Copilot
VS Code extension proxy
GitHub Copilot mode routes requests through a local VS Code proxy.

Setup: Install the KaliAI Copilot Bridge VS Code extension, then start the proxy:
code --install-extension kaliai-copilot-bridge
The extension starts a proxy on http://localhost:3002 that forwards requests to GitHub Copilot using your organization's models.
